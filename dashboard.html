🗄️ Database Schema

USERS TABLE: ├─ user_id (PK, SERIAL) ├─ username (UNIQUE, VARCHAR) ├─ password_hash (VARCHAR) ├─ salt (VARCHAR) ├─ created_at (TIMESTAMP) ├─ last_login (TIMESTAMP) ├─ account_locked (BOOLEAN) └─ locked_until (TIMESTAMP) VAULT_ENTRIES TABLE: ├─ entry_id (PK, SERIAL) ├─ user_id (FK → users) ├─ service_name (VARCHAR) ├─ service_username (VARCHAR) ├─ service_password (TEXT) ├─ notes (TEXT) ├─ created_at (TIMESTAMP) └─ updated_at (TIMESTAMP) AUDIT_LOG TABLE: ├─ log_id (PK, SERIAL) ├─ user_id (FK → users) ├─ action (VARCHAR: LOGIN, ADD, UPDATE, DELETE, VIEW) ├─ entry_id (FK nullable → vault_entries) ├─ action_time (TIMESTAMP) └─ ip_address (VARCHAR)
